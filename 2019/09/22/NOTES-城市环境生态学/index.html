发表于 2019-09-22 更新于 2019-10-12 分类于 课程笔记

Chap1 导论

1.1 城市环境生态学的概念

  • 环境生态学是研究人为干扰下，生态系统内在的变化机理、规律和对人类的反效应，寻求受损生态系统恢复、重建和保护对策的科学，即运用生态学理论，阐明人与环境间的相互作用及解决环境问题的生态途径。
  • 【环境生态学的研究内容】
    • 在人为干扰下生态系统内在变化机理和规律
    • 生态系统受损程度的判断
    • 生态系统的功能及保护
    • 解决环境问题的生态对策
  • 城市环境生态学以整体的观点，把城市视作一个以人为中心的生态系统，在理论上着重研究其发生和发展的原因、组合和分布的规律、结构和功能的关系、调节和控制的机理；其应用目的在于运用生态学原理规划、建设和管理城市，提高资源利用效率，改善系统关系，增强城市活力，使城市生态系统沿着有利于人类利益和可持续的方向发展。

1.2 环境极其结构

  • 环境科学所研究的环境，其主体是人类，环境指的是人类的生存环境，它的含义可以概括为：作用于人的一切外界事物和力量的总和。
  • 环境包括自然环境和人工环境。
    • 自然环境是人类出现之前就存在的，是人类目前赖以生存的自然条件和自然资源的总称，是直接或间接影响到人类的一切自然形成的物质、能量和自然现象的总体，它对人类的影响是根本性的。
    • 人工环境从狭义上讲是人类根据生产、生活、科研、文化、医疗等需要而创建的环境空间。
  • 从人类和环境相互作用的角度，由近及远、由小到大可分为聚落环境、地理环境、地质环境和星际环境。
    • 地理环境的含义是围绕人类的自然现象的总体。地理环境位于地球的表层，包括岩石圈、土壤圈、水圈、大气圈。
    • 地质环境指的是地理环境中除去生物圈以外的其余部分。

1.3 环境要素及属性

  • 环境要素，又称环境基质，是构成人类生存环境整体和各个独立的、性质不同而又服从整体演化规律的基本物质组分。
  • 环境要素可分为自然环境要素和人工环境要素。其中自然环境要素通常指水、大气、生物、岩石、土壤等。
  • 【基本属性】
    1. 最差（小）因子限制律
    整体环境的质量，不能由环境诸要素的平均状态决定，而是受环境诸要素中那个与最优状态差距最大的要素所控制；
因此在改进环境质量时，必须对环境诸要素的优劣状态进行数值分类，遵循由差到优的顺序依次改进，使之均衡地达到最佳状态。
    2. 等值性
    各个环境要素，无论它们本身在规模或数量上如何不同，但只要是一个独立的要素，那么对于环境的限制作用并无质的差异。
    3. 整体性大于各个体之和
    环境诸要素互相联系，互相作用产生的整体效应，是在个体效应基础上的质的飞跃。
    4. 互相联系及互相依存
    环境诸要素在地球演化史上的出现，有先后之别，但它们又是相互联系、相互依存的。从演化的意义上看，某些要素孕育着其他要素。

1.4 环境的功能与特性

-【环境的功能】
1. 空间功能。 指环境提供的人类和其他生物栖息、生长、繁衍的场所，且这种场所是适合其生存发展的。
2. 营养功能。 包含环境提供的人类和其他生物生长、繁衍所必需的各类营养物质，以及各类资源、能源。
3. 调节功能。 各类环境要素皆具有吸收、净化污染物，使受到污染的环境得到调节、恢复的能力。

  • 【环境自身特性】
    1. 环境平衡。环境系统内外存在着物质和能量的变化与交换；在一定的时空尺度内，若系统的输入等于输出，就出现平衡，称作环境平衡。
    2. 复杂性导致稳定性系统的组成和结构越复杂，它的稳定性就越大，越容易保持平衡。
    3. 子系统的协同作用。环境的各子系统和各组成成分之间，存在着复杂的相互作用，构成一个网络结构，使环境具有整体功能，形成集体效应，起着协同作用。
  • 【环境对于干扰所具有的特性】
    1. 整体性。人类环境的各组成部分之间存在着相互联系、相互制约的关系，局部地区的环境污染或破坏，总会对其他地区造成影响和危害。
    2. 有限性。地球的空间是有限的；且人类生存环境有其自身的有限性，如资源有限、稳定性有限、容纳污染物的能力有限或者说对污染物质的自净能力有限。
    3. 自净性。（环境本底值、环境容量）污染物质和污染因素进入环境后，将引起一系列物理的、化学的和生物的变化，使环境达到自然净化。
    4. 不可逆性。人类的环境系统在其运转过程中，能量流动的过程不可逆；所以环境一旦遭到破坏，利用物质循环规律，可以实现局部的恢复，但不能彻底回到原来的状态。
    5. 隐显性。日常的环境污染与环境破坏对人们的影响，其后果的显现，需要经过一段时间，要有一个过程。
    6. 持续性。环境污染不但影响当代人的健康，而且还可能会造成世世代代的遗传隐患。
    7. 灾害放大性。某方面不引人注目的环境污染与破坏，经过环境的作用以后，其危害性或灾害性，无论从深度和广度，都会明显放大。

Chap2 生态学基础

2.1 概述

  • 生态学是研究生物之间、生物与环境之间相互关系及其作用机理的科学。

2.2 生物生存环境——生物圈

  • 生物圈是地球上所有生物及其生活环境的总称。是囊括地球表面 岩石圈上层、大气圈下层以及水圈 的全部所有有生命存在的区域，由生物与非生物的物理化学环境组成的高度复杂的生态系统。
  • 【组成】
    • 生命物质：是生物有机体的综合。包括蛋白质、糖类、脂质核酸。
    • 生物生成性物质：是由生命物质所组成的有机矿物质相互作用的生成物，如煤、石油、泥炭和土壤腐殖质等。
    • 生物惰性物质：指大气底层的气体、沉积岩、黏土矿物和水。
  • 【范围】
    • 平流层下层
    • 对流层
    • 水圈
    • 岩石圈上层
  • 【特征】
    • 生物圈是地球上人类和生物的唯一生存地。
    • 生物圈具有很强的“生物化”特征，是一个极其复杂的、能自动调节的生命物质和非生命物质系统，积累和重新分配巨大的物质资源和能量资源。
    • 生物有机体呈现种类的多样性；生物多样性是生物圈的基本特征之一。
    • 生物圈的结构呈现不平衡性和不对称性。
    • 生物圈通过物质循环和能量转化来自我调节和平衡。

2.3 生态因子及其作用

  • 生境指在一定时间内对生命有机体生活、生长发育、繁殖以及有机体存活量有影响的空间条件及其他条件的总和。
  • 在生境中对生物的生命活动起直接作用的那些环境要素称为生态因子，如温度、湿度、土壤、食物、氧气等。
  • 小环境指对生物有着直接影响的邻接环境；大环境则指地区环境、地球环境和宇宙环境。
  • 影响生物生存的非生物因子常常是在相当大的地理区域内起作用的。
  • 【分类】
    • 生物因子：有机体（同种和异种）；非生物因子：温度、光、湿度、pH、氧气等。
    • 密度制约因子：食物、天敌等生物因子；非密度制约因素：温度、降水、气候等非生物因子
  • 【一般特征】
    1. 综合作用
      生物在一个地区生长发育，它所受到的环境因素影响不是单因子的，而是综合的；这些因子彼此联系、互相促进、互相制约，任何一个因子的变化，必将引起其他因子不同程度的变化，且生态因子之间在一定条件下是可以互相转化的。
    2. 主导因子作用（非等价性）
      某一因子的变化会引起其它因子发生明显的变化或能够明显高边生物的生长和发育。
    3. 生态因子的不可替代性和补偿作用
      生态因子对生物的作用各不相同，从总体上来说生态因子是不可替代的，但在局部是可以作一定的补偿。
    4. 生态因子的直接作用和间接作用
      生态因子对生物的生长、发育、繁殖及分布的作用可分为直接作用和间接作用。
    5. 因子作用的阶段性（限定性）
      生物在生长发育的不同阶段往往需要不同的生态因子或生态因子的不同强度。
  • 【作用规律】
    1. 限制因子规律
      在环境诸因子中，某个因子限制了生物的生长、发育、繁殖或生存，则称该因子为限制因子。
    2. 最低量（最小因子）定律
      “水桶定律”；
      作物的产量常常不是被需要量大的营养物质所限制，而是受某些微量元素所限制。
    3. 耐受性定律
      • 对于具体的生物, 各种生态因子都存在着一个生物学的上限和下限(又称阈值), 它们之间的幅度就是该种生物对某一生态因子的耐性限度, 其中包括最适区, 适宜区和高低死亡限(见耐受曲线)。
      • 在自然界，由于长期自然选择的结果，每个种都适应于一定的生境，并有其特定的适应范围。每个种适应生境范围的大小即生态幅。生态幅就是生物对生态因子的适应性结果的反映。

2.4 生态系统的基本概念及类型

  • 【概念】生态系统是指在一定的时间和空间内，生物和非生物成分之间，通过物质循环、能量流动和信息传递，而相互作用、相互依存所构成的统一体，是生态学的功能单位。
  • 【共同特性】
    • 生态系统是生态学上的一个主要结构和功能单位，属于生态学研究的最高层次。
    • 生态系统内部具有自我调节能力。
    • 能量流动、物质循环和信息传递是生态系统的三大功能。
    • 营养级的数目通常不会超过5~6个。
    • 生态系统是一个动态系统。
  • 【组成】
    • 非生物环境：无机物质；有机化合物；气候因素
    • 生物成分：生产者；消费者；分解者
  • 【基本特征】
    • 生态系统具有生物学特征。
    • 生态系统具有一定的区域特征
    • 生态系统是开放的自律系统
    • 生态系统是一种反馈系统
  • 【结构】
    • 形态结构
      生物种类、数量的空间配置和时间变化，即生态系统的空间结构和时间结构，如森林生态系统的垂直分布，季相等。
    • 营养结构
      1. 食物链：植物所固定的能量通过一系列的取食和被取食关系在生态系统中传递，各种生物由于营养关系而形成的链状顺序称为食物链。
      2. 食物网：生态系统中的食物链彼此交错连接，形成一个网状结构，这就是食物网。
  • 【类型】
    • 按人类对生态系统的干预程度划分：自然生态系统；半自然生态系统；人工生态系统
    • 按生态系统空间环境性质划分：陆地生态系统；淡水生态系统；海洋生态系统
